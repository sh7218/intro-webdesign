C++

General Information: C++ is a programming language published in 1983 by Villain Stroustrup, a Danish computer scientist, of Bell Labs in the United States as an extension of the C programming language. It was designed to provide an object-oriented programming paradigm while still maintaining the efficiency and low-level control of C. It is a procedure-oriented language that values the order of program execution and is characterized by supporting structured programming and modular design concepts. It also has many advantages such as object-oriented and multi-paradigm, and is used especially in game development.
Comparing with Java: Similar to Java, C++ is a compiled language that has to be compiled into machine code before it can be executed.
